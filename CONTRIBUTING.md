# Contributing

Thank you for your interest in contributing to this project!

## How to Contribute

1. Fork the repository on GitHub.
2. Create a new branch for your changes:
   ```bash
   git checkout -b my-feature
   ```
3. Make your changes and commit them with a clear message:
   ```bash
   git commit -m "Add: brief description of your change"
   ```
4. Push to your fork and open a Pull Request against the `main` branch.

## Code Style

- Keep changes focused and minimal.
- Add comments where the intent is not immediately obvious.
- Update `README.md` if your change affects setup or usage instructions.

## Reporting Issues

Please open a [GitHub Issue](https://github.com/MatthewOscar/GitHub-to-GitLab/issues) for bugs or feature requests.
